# BioCpp
Bioinformatics Library for C++

## Installation
```bash
$ cmake CMakeLists.txt -B {folder to build} (-G "MinGW Makefiles")
$ make
$ {folder to build}/biocpp_test(.exe)
```
See [WIKI](https://github.com/panda5176/biocpp/wiki) for more information
